# Portfolio Case Study Draft

## Problem

Job seekers apply to many roles, but manually tailoring resumes and cover letters takes time. Generic applications often perform poorly.

## Solution

This project uses AI to help job seekers find relevant roles, compare job descriptions to their resume, and create tailored application materials with human approval.

## Technical Highlights

- Full-stack app using React and FastAPI
- PostgreSQL-backed application tracker
- AI-powered job matching and resume suggestions
- Agent-style workflow with clear safety controls
- Deployment-ready architecture

## Demo Plan

1. Upload resume
2. Add job posting URL or text
3. Show match score
4. Generate tailored resume bullets
5. Generate cover letter draft
6. Save application in tracker
